### PROMPT TO GENERAGE design-brief ###

Design a complete, production-ready mobile UI kit for a Healthcare Service Request Tracking application. The app allows users to create, track, and manage healthcare service requests such as appointments, exams, medication, billing, and general support.
The application is offline-first, meaning every action is applied instantly and synchronized later. The interface must clearly communicate synchronization states (Pending Sync, Synced, Failed) without creating unnecessary user anxiety.
Design for 390 × 844 px (iPhone 15), compatible with both iOS and Android (Flutter). The visual style should be modern, minimal, trustworthy, healthcare-oriented, with generous whitespace, rounded corners, soft shadows, and accessibility (WCAG AA). All UI copy must be in Brazilian Portuguese (pt-BR), while component and layer names should be in English.
